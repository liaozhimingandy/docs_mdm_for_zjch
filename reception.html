主数据系统接收数据

描述

业务系统将主数据成员推送到主数据管理系统接口服务说明

请求方式
名称 说明
协议 SOAP
方式 Web Service
地址 详情请点击此链接
方法 PushMember
请求参数
名称 必选 数据类型 长度 说明
system_id 是 String 100 业务系统ID;主数据系统获取
data 是 String 不限 主数据成员xml字符串

data成员节点说明

节点 说明 类型 是否为空 备注
MASTER_MEMBER 根节点
MASTER_DEF_ID 主数据定义主键 String 否 字典表id;该字段值为主数据定义分类id;从主数据管理系统获取;
CODE 成员编码 String 否
NAME 成员名称 String 否
GENDER 性别 String 自定义 定义
AGE 年龄 Int 自定义 主数据自定义
…………… 其他自定义属性 自定义 自定义 主数据自定义
返回数据
节点 说明 备注
errorcode 错误代码 200：推送成功 400：推送失败
message 结果说明
resultdata 返回数据 若新增则返回值为主数据成员id;反之返回对应的修改说明

返回示例

<ResultType>
    <errorcode>200</errorcode>
    <message>推送成功(新增)！</message>
    <resultdata>230084F9DBF84C3F93E5850C5BE637A2</resultdata>
</ResultType>
Soapui示例

请求数据

<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/" xmlns:car="http://www.caredigm.com/">
   <soapenv:Header/>
   <soapenv:Body>
      <car:PushMember>
         <!--Optional:-->
         <car:system_id>156AA90DB75C49A6A5D9395FF7CCF16B</car:system_id>
         <!--Optional:-->
         <car:data><![CDATA[   <MASTER_MEMBER>><MASTER_DEF_ID>885ee5f66eb5469e983f0e3d3795b5c5</MASTER_DEF_ID><CODE>1</CODE>    <NAME>男</NAME><DESC>1235</DESC></MASTER_MEMBER>  ]]></car:data>
      </car:PushMember>
   </soapenv:Body>
</soapenv:Envelope>

返回数据

<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns:xsd="http://www.w3.org/2001/XMLSchema">
   <soap:Body>
      <PushMemberResponse xmlns="http://www.caredigm.com/">
         <PushMemberResult><![CDATA[<ResultType><errorcode>200</errorcode><message>推送成功(新增)！</message><resultdata>230084F9DBF84C3F93E5850C5BE637A2</resultdata></ResultType>]]></PushMemberResult>
      </PushMemberResponse>
   </soap:Body>
</soap:Envelope>

[!danger]

<![CDATA[#此处为数据内容#]]>:修饰内容为不被xml解析器解析
