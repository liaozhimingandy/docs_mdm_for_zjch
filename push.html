主数据系统推送数据

描述

主数据管理系统将成员推送到ESB集成平台接口服务说明

请求方式
名称 说明
协议 RESTful
方式 POST
地址 url地址由ESB平台提供
请求参数
必选 数据类型 长度 说明
是 String 不限 主数据成员xml字符串
成员节点说明
节点 说明 类型 是否为空 备注
MASTER_MEMBER 根节点
MASTER_MEMBER_ID 主数据系统成员主键 String 否
CODE 成员编码 String 否
NAME 成员名称 String 否
GENDER 性别 String 自定义 定义
AGE 年龄 Int 自定义 主数据自定义
…………… 其他自定义属性 自定义 自定义
PY 拼音简码 String 否 由NAME值生成
STATUS 状态 String 否 1-新建；2-维护中；3-已提交待审核；4-已审核待整改；5-已审核待发布；6-已发布
DELETE_FLAG 删除状态 String 否 0-未删除；1-已删除
IS_ENABLE 0-禁用；1-启用
CREATOR_ID 创建人id String 是
CREATOR_NAME 创建人姓名 String 是
CREATE_TIME 创建时间 Datetime 是
LAST_MODIFIER_ID 最后修改人id String 是
LAST_MODIFIER_NAME 最后修改人姓名 String 是
LAST_MODIFY_TIME 最后修改时间 Datetime 是  
推送示例
<MASTER_MEMBERS>
  <SYSTEMS>
    <SYSTEM>
      <SYSTEM_ID>156AA90DB75C49A6A5D9395FF7CCF16B</SYSTEM_ID>
      <SYSTEM_CODE/>

      <SYSTEM_FIELDS>null</SYSTEM_FIELDS>
    </SYSTEM>
  </SYSTEMS>
  <MASTER_MEMBER>
   <MASTER_MEMBER_ID>4B2A3849D5B249289DFC72BBB99EC390</MASTER_MEMBER_ID>
    <LAST_MODIFIER_ID>1</LAST_MODIFIER_ID>
    <SORT>90</SORT>
    <MASTER_DEF_ID>e1ca5dd3664845bd8bac488f5905e828</MASTER_DEF_ID>
    <CREATOR_ID>1</CREATOR_ID>
    <CREATOR_NAME>系统管理员</CREATOR_NAME>
    <STATUS>6</STATUS>
    <LAST_MODIFY_TIME>2020-05-19 09:50:34</LAST_MODIFY_TIME>
    <NAME>未知</NAME>

</MASTER_MEMBERS>
